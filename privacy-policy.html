Privacy Policy for ReadTracker

1. 데이터 수집 방식 및 이유 설명

ReadTracker 확장 프로그램은 사용자가 저장한 제목을 로컬 스토리지 또는 chrome.storage.sync에 저장합니다. 수집된 데이터는 확장 프로그램의 기능을 제공하기 위해 사용됩니다. ReadTracker는 사용자의 동의 없이 어떠한 개인 식별 정보를 수집하거나 저장하지 않습니다.

2. 데이터 사용 방식 설명

저장된 제목 데이터는 사용자가 동일한 제목을 다시 볼 때 알림을 제공하는 데 사용됩니다. 이 데이터는 로컬 스토리지 또는 chrome.storage.sync에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 데이터 보안 설명

ReadTracker는 사용자 데이터의 보안을 중요하게 생각합니다. 저장된 데이터는 사용자의 브라우저 로컬 스토리지 또는 사용자 계정에 동기화되어 저장되며, 외부 접근으로부터 보호됩니다.

4. 사용자의 데이터 접근 권한 설명

사용자는 언제든지 확장 프로그램의 설정에서 저장된 제목 데이터를 삭제할 수 있습니다. 삭제된 데이터는 복구할 수 없습니다.

전용 목적

ReadTracker 확장 프로그램의 전용 목적은 사용자가 인터넷에서 동일한 콘텐츠를 여러 번 읽지 않도록 돕기 위해 제목을 저장하고 관리하는 것입니다. 이를 통해 사용자는 중복된 콘텐츠를 피하고 시간을 효율적으로 사용할 수 있습니다.

저장된 제목 데이터는 로컬 스토리지 또는 chrome.storage.sync에만 저장되며, 사용자가 동일한 제목을 다시 방문할 때 알림을 제공하는 데 사용됩니다. 수집된 데이터는 외부 서버로 전송되지 않으며, 사용자 동의 없이 제3자와 공유되지 않습니다.

activeTab 권한 사용 근거

ReadTracker 확장 프로그램은 activeTab 권한을 사용하여 사용자가 현재 활성화된 탭에서 제목을 읽고 저장할 수 있도록 합니다. 이 권한은 다음과 같은 목적으로 사용됩니다:

  • 현재 활성 탭의 제목을 읽기: 사용자가 현재 보고 있는 웹 페이지의 제목을 읽어와 로컬 스토리지 또는 chrome.storage.sync에 저장합니다. 이를 통해 사용자는 동일한 제목의 콘텐츠를 다시 방문할 때 알림을 받을 수 있습니다.
  • 사용자 경험 개선: activeTab 권한을 통해 현재 활성화된 탭에서 필요한 데이터를 즉시 처리하여 사용자가 확장 프로그램을 더 편리하게 사용할 수 있도록 합니다.

이 권한은 사용자가 확장 프로그램을 직접 활성화했을 때만 작동하며, 사용자의 브라우징 활동을 추적하거나 기록하지 않습니다. ReadTracker 확장 프로그램은 사용자 동의 없이 데이터를 수집하거나 외부로 전송하지 않습니다.

ReadTracker 확장 프로그램은 chrome.storage.sync API를 사용하여 저장된 제목 데이터를 여러 디바이스에서 동기화합니다. 이는 사용자가 Chrome 계정에 로그인한 상태에서 어떤 디바이스를 사용하든 동일한 제목 데이터를 접근하고 관리할 수 있도록 하기 위함입니다.

  • 데이터 동기화: chrome.storage.sync를 통해 사용자가 여러 디바이스에서 동일한 데이터를 유지할 수 있습니다. 이는 사용자가 동일한 제목의 콘텐츠를 다시 방문할 때 알림을 받을 수 있도록 합니다.
  • 사용자 경험 개선: chrome.storage.sync를 사용하여 데이터를 동기화함으로써, 사용자는 다른 디바이스에서도 일관된 경험을 유지할 수 있습니다. 이를 통해 중복된 콘텐츠를 피하고 시간을 효율적으로 사용할 수 있습니다.
  • 데이터 보안 및 프라이버시: ReadTracker는 사용자의 동의 없이 데이터를 수집하거나 외부로 전송하지 않습니다. 저장된 제목 데이터는 사용자 계정에 안전하게 동기화되며, 외부 접근으로부터 보호됩니다.
